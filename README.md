### Start development

```bash
$ cp .env.example .env

$ npm i

$ npm run start
```

### Build production

```bash
$ cp .env.example .env

$ npm i

$ npm run build:prod
```

### Lint

```bash
$ npm run lint

$ npm run lint:fix
```
